Chaos Theory

What is Chaos Theory?

Chaos theory is the study of dynamic systems which are highly sensitive to initial conditions (i.e one small change to the initial conditions can cause a completely different end result). These systems can be found both in human generated areas and naturally occurring ones. One specific area of naturally occurring chaos theory is weather, lightning in particular. Since the circumstances under which lightning is created are still not known to us, it is currently impossible to predict where lightning will strike, and in what frequency.

Above is an example of a fractal generated using chaos theory; it is known as the Sierpinski Triangle. It is generated using a simple method that, when first looked at, does not appear to accomplish anything special. Here is a step-by-step to list to the generation of the Sierpinksi Triangle using chaos theory:

  1. Start with a triangle ABC. It does not need to be equilateral or have any special rules, however triangles that are closer to equilateral look better in the end.
  2. Start with a point anywhere within the bounds of the triangle.
  3. Choose A, B, or C randomly, and take the midpoint of that vertex and the original point.
  4. Repeat this process for the new point, and all new points subsequently formed.
  5. Be amazed as this process actually generates you the Sierpinski Triangle.

It is a relatively simple program to implement. Here are the steps for what it does:

  1. Loops for variable "i" (the number of points that you want to create
  2. Runs the function random for[1,3] (which basically chooses a random INTEGER using a minimum of 1 and a maximum of 3
  3. Then, depending upon that random number, it takes the midpoint of current selected point and the randomly chosen vertex (the function "midpoint" being quite self explanatory)
  4. The program then draws a point at that midpoint, and switches the current point to that new points, and then iterates the process once again
